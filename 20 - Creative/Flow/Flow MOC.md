# Flow — Map of Content

> *"A tool that visualizes creativity itself — something that helps color flow like sound."*

Flow (IdeationBot) is a generative ideation assistant that helps designers move from a vague idea → structured creative concept → tangible output. Built on React + Gemini AI + Adobe Firefly.

## The Guiding Idea

Flow is not another chat interface. It's a **thinking environment** that blends creative psychology and AI reasoning. Where traditional brainstorming tools collect ideas, Flow **interprets relationships between them** — it maps conceptual topology.
