# Flow — UI & Design System

## Layout Structure

```
┌─────────────┬──────────────────────────┬─────────────┐
│    SEED     │        CANVAS            │   OUTPUTS   │
│   (Input)   │   (Generative Space)     │  (Results)  │
│ • Upload    │ • Ideation nodes         │ • Insights  │
│ • Describe  │ • Connection lines       │ • Prompts   │
│ • Drag files│ • Pulse animations       │ • Briefs    │
└─────────────┴──────────────────────────┴─────────────┘
│                    FLOW BAR                           │
│          (Session timeline + energy rhythm)           │
└───────────────────────────────────────────────────────┘
```

## Visual Theme: Synesthetic Gradients

> *"Color flows like sound"*

- Color behaves like audio waves — it moves, bleeds, and resonates
- Gradients shift based on ideation energy/activity level
- Motion design mirrors the psychology of creative flow state
- Typographic neutrality — personality comes from color, motion, scale, and spacing

## Typography

- High-legibility HCI fonts: Inter, IBM Plex, or Satoshi
- Let type disappear into the work, not compete with it

## Motion Design Principles

| Moment | Animation Type | Purpose |
|--------|---------------|---------|
| Node generation | Pulse + scale-in | Shows new thought arriving |
| Branch connection | Draw-on line | Maps relationship forming |
| Remix trigger | Color wave | Signals divergent thinking |
| Flow Mode entry | Chrome fade | Signals immersion |
| Idle state | Breathing loop | Ambient calm presence |

## Flow Mode (Deep Work State)

- 95% of UI chrome hidden
- Keyboard-driven workflow
- Breathing animations + optional soundscapes
- Progressive disclosure — interface teaches itself into invisibility

## Color Rhythm System

- Cool blues/purples → exploratory / divergent thinking
- Warm ambers/corals → synthesis / convergent thinking
- Neutral grays → resting state
- Electric cyan/lime → breakthrough / insight moment
