# Design References

> Visual references, aesthetic territories, and work that has influenced your thinking.

---

## Aesthetics you've worked in

### IWD 2026 — Protest print
Interview Magazine meets 90s protest zine. Bold type, halftone textures, deep purple + forest green + white + yellow accent. No pink. No gradients. No soft shadows. Suffragette palette. Misregistration, grain, duotone.

### The Lookline — Cool older sibling
Clean, warm, direct. Unbounded headlines (short and punchy). Montserrat body. Deep teal `#003549` with light pink `#F19484` only as accent — never as a primary action color. Feels like a real person said it, not a brand manager.

### CodeQuest — Terminal
Dark card, traffic light dots (red/yellow/green), JetBrains Mono. Syntax-highlighted code. Green/red feedback bars. Feels like a real IDE, not an edtech product.

---

## References worth revisiting

- **Interview Magazine** — editorial hierarchy, image/type tension
- **90s protest print** — energy, imperfection as intent, political legibility
- **Dieter Rams** — "Good design is as little design as possible" — still the benchmark for purposeful restraint

---

## Notes on visual language

Aesthetics aren't decoration — they're arguments. The IWD project used protest print language because the content demanded it. The Lookline uses warmth because the product is about trust. CodeQuest uses terminal aesthetics because the audience should feel like they're in a real environment.

Ask what the visual language is *saying*, not just what it looks like.
