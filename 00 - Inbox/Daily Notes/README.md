# Daily Notes

> Daily notes land here automatically if you configure Obsidian's Daily Notes plugin to use this folder.

**To set this up:**
Settings → Core Plugins → Daily Notes → New file location → `00 - Inbox/Daily Notes`

Until then, any daily notes created by the plugin will land at the vault root. Move them here or trash them from there.

---

*This folder is a container — no content lives here directly.*
